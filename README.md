Simple Text Editor
Installation
1. Ensure Tkinter is installed:
For Ubuntu/Debian:


sudo apt-get update
sudo apt-get install python3-tk


For Fedora:


sudo dnf install python3-tkinter


For Arch Linux:


sudo pacman -S tk


For macOS with Homebrew:


brew install python-tk
2. Run the program:
        python mytexteditor.py
